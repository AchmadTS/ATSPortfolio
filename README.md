## ⚙️ Instalasi

### 1. Clone repository

```bash
git clone https://github.com/Achmadts/ATSPortfolio.git
cd ATSPortfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Jalankan frontend

```bash
npm run dev
```

### 4. Jalankan backend chat server

```bash
npm run start
```

---

## 🔐 Environment Variables

Buat file `.env` di root project, lalu isi seperti berikut:

```env
OPENROUTER_API_KEY=isi_api_key_kamu_di_sini
VITE_IS_DEPLOYMENT=false
```

### Keterangan:

* `OPENROUTER_API_KEY` digunakan untuk fitur chat copilot
* `VITE_IS_DEPLOYMENT` digunakan untuk membedakan environment deployment dan local
